# GuardIA — Web pública

Landing y demo pública de **GuardIA**, la herramienta moderna para coordinar guardias hospitalarias.

🌐 **En vivo**: [miguardia.es](https://miguardia.es)

## Estructura

- `/` — Landing principal (`miguardia.es`)
- `/demo` — Demo pública del motor de asignación (`miguardia.es/demo`)

## Stack

- HTML estático + CSS + JS vanilla
- React 18 (CDN, solo en la demo)
- SheetJS (CDN, para exportar Excel en la demo)
- Tipografía: DM Serif Display + DM Sans
- Hosting: Vercel
- Dominio: cdmon

## Desarrollo local

No requiere build. Abre `index.html` directamente en el navegador, o usa un servidor estático simple.

GuardIA es un proyecto en construcción. Beta privada prevista para primavera 2026.
